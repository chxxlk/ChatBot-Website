* *AI        : Gemini AI*

# 💡 Installation
```
# Clone repo
git clone https://github.com/chxxlk/Chatbot-Website.git

# Masuk Foleder backend
cd gemini-chatbot-backend
composer install
cp .env.example .env
# Tambahkan Databse, Gemini API, Cors, dll di .env

# Masuk folder frontend
cd gemini-chatbot
npm install
```
# ▶️ Usage
```
# Jalankan backend
php artisan serve

# Jalankan frontend
npm run dev
```
